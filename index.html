What is iShamu?

iShamu generates in game currency and automatic wins in a variety of server based games, without the need to modify game files.

Add repo: https://ishamu.fun

WHEN PURCHASING ENTER YOUR ACTIVATION ID

This is used to ensure that your Device is activated with a licence to use iShamu. How to get your Activation ID? Your Activation ID can be found when you iShamu and select a game. If you have any questions join the Discord channel linked above. License will last for a year.
